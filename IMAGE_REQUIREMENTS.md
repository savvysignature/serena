## Image Guidelines

- All images should be optimized for web (compressed, but high quality)
- Use JPG format for photos, PNG for images with transparency
- Ensure images are properly sized to avoid loading issues
- Consider using professional photography for Serena's images
- Background images should be subtle and not distract from content
- Maintain consistent color tones across images for a cohesive look

## Color Scheme Reference

The website uses a modern professional color palette:
- Primary: Indigo (#6366f1)
- Secondary: Purple (#8b5cf6)
- Accent: Pink (#ec4899)
- Neutral backgrounds: Gray tones (#fafafa, #f5f5f5)

Images should complement this color scheme.
